ORGANIC BUILDER

Motto

"Make up our own rules of construction to simulate open-ended, creative evolution. Forget all that chemistry they made you learn in school, we're going to make it all up as we go along. "

Short Description

Java simulation of evolution -the process in which natural selection and mutation can lead to complex, sophisticated design- using artificial chemistry.
