Get it, try it

This software has been built using the java language. In order to successfully run it, you will need to have a Java Virtual Machine installed, 1.8.0+ compliant. Check now your browser support.

Being packaged as *.jar, the software can either be launched

  • with Java Web Start (What is Java Web Start?)
  • as an Applet : (What is an applet?)
  • by downloading the latest release and double-clicking on it
  • by downloading the latest release and typing in your shell java -jar downloadedJarFileName

Launch right now using

  • Java Web Start : in english or in french
  • Applet : in english or in french

If you want to look at the source codes, they can be found in the latest release and at github.
