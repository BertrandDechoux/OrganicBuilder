> To run this application type : java -jar OrganicBuilder.jar
This will start the application using the default language (english). You may want to choose another language by using a valid ISO-639 Language Code (more information on http://www.loc.gov/standards/iso639-2/php/English_list.php). E, for french, you will type : java -jar OrganicBuilder.jar fr 
Currently, only french is available. You are welcome to translate the Organic Builder to any another language.
> To display this help use : java -jar OrganicBuilder.jar -help
> To display information about this application : java -jar OrganicBuilder.jar -about
> To display the full license of this application : java -jar OrganicBuilder.jar -license
